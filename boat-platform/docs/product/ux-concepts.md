# UX Concepts

## Web Dashboard Concepts

- **Signal Timeline View:** Zoomable time-series view per signal, similar to performance tooling workflows.
- **Scenario Editor:** YAML editor with live validation feedback using JSON Schema.
- **Plugin Registry:** Card-based registry with plugin metadata, health, and compatibility state.
- **Simulation Control Bar:** Play, pause, step, reset, stop actions with tick counter and wall-time indicator.
- **Fault Injection Panel:** Timeline-based fault scheduling with drag-and-drop editing.
- **Trace Browser:** Table view of traces with export (MF4/CSV) and replay controls.

## CLI UX Concepts

- Predictable command groups: `boat sim`, `boat scenario`, `boat replay`, `boat plugin`
- Consistent output modes: human-readable tables and machine-readable JSON
- Explicit error messages with actionable next steps

## Accessibility and Usability

- Keyboard-first navigation for dashboard control surfaces
- High-contrast chart and state indicators
- Error states surfaced with clear remediation guidance
